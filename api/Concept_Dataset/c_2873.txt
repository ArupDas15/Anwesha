দ্য বাস্কেটবল ডায়েরিজ ১৯৯৫ সালে মুক্তিপ্রাপ্ত মার্কিন সমকালীন ক্রাইম ড্রামা চলচ্চিত্র। লেখক জিম ক্যারলের একই নামের আত্মজীবনীমূলক গ্রন্থ অবলম্বনে ছবিটি পরিচালনা করেছেন স্কট কালভার্ট। লেখক জিম ক্যারলের কৈশোর বয়সে বাস্কেটবল খেলোয়াড় ও পরে লেখক হিসেবে নিজেকে উপস্থাপন এবং বাজে বন্ধুদের পাল্লায় পড়ে হিরোইনে আসক্ত হওয়া দেখানো হয়েছে। এতে জিম ক্যারলের ভূমিকায় অভিনয় করেছেন লিওনার্দো ডিক্যাপ্রিও। এছাড়া অন্যান্য ভূমিকায় রয়েছেন লরেন ব্রাকো, জেমস মাদিও, মার্ক ওয়ালবার্গ প্রমুখ।জিম ক্যারলের মূল বই দ্য বাস্কেটবল ডায়েরিজ ১৯৬০-এর দশকে লেখা হয় এবং ১৯৭৮ সালে প্রকাশিত হয়। কিন্তু চলচ্চিত্রটি ১৯৯০-এর দশকের আদলে নির্মিত হয়েছে।
