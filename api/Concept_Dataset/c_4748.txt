একজন ইন্টারনেট উদ্যোক্তা একটি ইন্টারনেট ভিত্তিক ব্যবসায়ের মালিক, প্রতিষ্ঠাতা বা পরিচালক। এই তালিকায় রয়েছে ইন্টারনেট ভিত্তিক কোম্পানির প্রতিষ্ঠাতা, এমন ব্যক্তিরা যাদের ব্যবসায়িক দক্ষতার জন্য তাদের বিভিন্ন কোম্পানিতে নিয়ে আসা হয়েছে, যেমন উদ্যোক্তাদের দ্বারা চালিত কোম্পানিগুলোতে নিয়োগকৃত সিইও।
