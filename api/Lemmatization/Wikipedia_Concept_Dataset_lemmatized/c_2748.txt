রুশ ফুটবল ইউনিয়ন রুশঃ Российский Футбольный Союз রসিস্কি ফুতবলনি সয়ুজ ইংরেজিঃ euresian Football Union এছাড়াও সংক্ষেপ আরএফএস নামে পরিচিত রাশিয়া ফুটবল সর্বোচ্চ নিয়ন্ত্রক সংস্থা সংস্থা ১৯১২ সাল ১৯শে জানুয়ারি তারিখে প্রতিষ্ঠিত প্রতিষ্ঠা বছরে ফুটবল সর্বোচ্চ সংস্থা ফিফা সদস্যপদ লাভ পাশাপাশি প্রতিষ্ঠা ৪২ বছর ১৯৫৪ সাল সংস্থা আঞ্চলিক সংস্থা উয়েফা সদস্যপদ লাভ সংস্থার সদর দপ্ত রাশিয়া রাজধানী মস্কো অবস্থিত সংস্থা রাশিয়া পুরুষ নারী অনূর্ধ্ব ২৩ দল পাশাপাশি ঘরোয়া ফুটবল রুশ প্রিমিয়া লীগ রুশ নারী ফুটবল চ্যাম্পিয়নশিপ রুশ কাপ প্রতিযোগিতা কার্যক্রম পরিচালনা আরএফএস রুশ ফুটবল প্রিমিয়া লীগ রাশিয়া বেশিরভাগ ফুটবল লীগ রেফারি ফুটবল টুর্নামেন্ট নিষেধাজ্ঞা প্রদান বর্তমানে রুশ ফুটবল ইউনিয়ন সভাপতি দায়িত্ব পালন আলেসাদ্ দুকাফ সম্পাদক দায়িত্ব পালন আলেক্সান্দ্ আলায়েভ
