১৯৭০ সাল তীব্র ভোলা ঘূর্ণিঝড় শক্তিশালী ক্রান্তীয় ঘূর্ণিঝড় ১৯৭০ সাল ১৩ই নভেম্বর তৎকালীন পূর্ব পাকিস্তান বর্তমান বাংলাদেশ দক্ষিণাঞ্চল আঘাত হান রেকর্ডকৃত ঘূর্ণিঝড়সমূহ সবচেয়ে ভয়াবহ ঘূর্ণিঝড় সর্বকাল সবচেয়ে ভয়ঙ্করতম প্রাকৃতিক দুর্যোগ ঝড় কারণে ৫ লাখ মানুষ প্রাণ হারায় অধিকাংশই গাঙ্গে বদ্বীপ সমুদ্র সমতল ভূমিত জলোচ্ছ্বাস ডুবে মারা ১৯৭০ সাল ভারতী ঘূর্ণিঝড় মৌসম ৬ষ্ঠ ঘূর্ণিঝড় মৌসম সবচেয়ে শক্তিশালী ঘূর্ণিঝড় সিম্পসন স্কেল ৩য় মাত্রা ঘূর্ণিঝড়
