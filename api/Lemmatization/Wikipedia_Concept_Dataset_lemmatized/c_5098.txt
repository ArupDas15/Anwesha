বাংলানিউজটোয়েন্টিফো কম বাংলাদেশী অনলাইন দৈনিক সংবাদ সংস্থা বাংলানিউজ বাংলা ভাষা ২৪ ঘণ্টা সার্বক্ষণিক খবর প্রকাশ বাংলা ওয়েব পোর্টাল সংবাদ বিনোদন সারাক্ষণ স্লোগান ২০১০ সাল ১ জুলাই বাংলানিউজ যাত্রা প্রতিষ্ঠাকাল পোর্টালটি প্রধান সম্পাদক দায়িত্ব প্রখ্যাত সাংবাদিক আলমগী হোসেন রাজধানী ঢাকা বাংলাদেশ প্রত্যেক জেলা প্রতিনিধি যুক্তরাষ্ট্ যুক্তরাজ্য কানাডা লন্ডনসহ বিশ্ব বড় বড় শহরেও রয়েছেন বাংলানিউজ নিজস্ব প্রতিনিধি
