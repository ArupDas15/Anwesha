সিরাজগঞ্জ এনায়েতপু সড়ক সিরাজগঞ্জ শহর বেলকুচি এনায়েতপু সংযোগ শাহজাদপু সড়ক সড়ক বেলকুচি এনায়েতপু চৌহালি শাহজাদপু পাবনা অঞ্চল মানুষ সিরাজগঞ্জ দেশ জেলা যাতায়েত প্রধান সড়ক সড়কটিত বাস মিনিবাস মাইক্রোবাস ট্রাক পিকআপ মোটরসাইকেল বেবীটেক্সী অটোরিকশা ভ্যন ফিটনেস ফিটনেস বিহিন গাড়ি চলাচল সম ঘটে দুর্ঘটনা ২০ কিলোমিটার সড়কটিত ভারী ধীর গতি হালকা যানবাহন চলাচল কারা লাইন হাত নিলেও শেষ সয়দাবাদ উচ্চ বিদ্যাল
