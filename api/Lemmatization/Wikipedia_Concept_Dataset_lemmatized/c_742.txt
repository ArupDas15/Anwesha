লাদাখি ভাষা তিব্বতিঃ ལ་དྭགས་སྐད་ ওয়াইলিঃ La dwags kat বোধি ভাষা ভারত লাদাখ অঞ্চল ব্যবহৃত তিব্বতি ভাষা জম্মু কাশ্মীর রাজ্যের লেহ জেলার প্রধান ভাষা এটিকে পার্শ্ববর্তী কার্গিল জেলা পুরিগি বালতি তিব্বতি ভাষা পরিবার অন্তর্গত ভাষা পরিবার তিব্বতি ভাষার পরস্পর অবোধগম্য ভারত ২ ০০ ০০০ লাদাখি বক্তা আছেন চীন তিব্বত স্বায়ত্তশাসিত অঞ্চল সম্ভবত ১২ ০০০ বক্তা বসবাস অধিকাংশ চাংথাং অঞ্চল সম্প্রদা ভাষার ইতিবাচক মনোভাব বয়েস ভাষার সক্রিয় সত্ত্বেও লাদাখি মাতৃভাষীদ সংখ্যা ২০০১ সাল ১ ০০ ০০০ ২০১১ সাল ১৪ ০০০ তে হ্রাস পেয়েছে
