পেপারমিন্ট লিনাক্স ওএস Peppermint Linux OS স্টেবল ক্লাউড কেন্দ্রিক উবুন্টু ভিত্তিক হালকা লিনাক্স ডিস্ট্রিবিউশন পেপারমিন্ট লিনাক্স ওএসে ডেস্কটপ পরিবেশ এলএক্সডিই এক্সসিই শংকর সংস্করণ কম হার্ডওয়্যা রিসোর্স লাগে পুরোনো যন্ত্র ডিস্ট্রো উপযোগী আইস এপ্লিকেশন ওয়েব ভিত্তিক এপ্লিকেশন তৈরি ব্যবস্থাপনা হাতিয়া ডিস্ট্রিবিউশন এটিকে স্বকী তুলেছে পূর্ণ ডেস্কটপ অভিজ্ঞতা ক্লাউড কম্পিউটিং এটাকে ক্রোম এস যোগ্য প্রতিদ্বন্দ্বী শিক্ষানবিশ লিনাক্স ব্যবহারকারীদ সহজ পরিচিত ডেস্কটপ পরিবেশ দেয়া অল্প রিসোর্সেও চলতে বিষয় নিশ্চিত ডেভলপারদ মূলনীতি
