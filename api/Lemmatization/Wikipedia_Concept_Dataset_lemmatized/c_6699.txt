রাসায়নিক বিক্রিয়া গতিত সংঘটিত বিক্রিয়া হার বিক্রিয়া গতি বিক্রিয়া হা একক সম উৎপাদ ঘনমাত্ বৃদ্ধি একক সম বিক্রিয়ক ঘনমাত্ হ্রাস সমানুপাতিক সংজ্ঞায়িত বিক্রিয়া হার নাটকীয়ভাবে পরিবর্তিত উদাহরণস্বরূপ পৃথিবীর বায়ুমণ্ডলীয় পরিবেশ লোহার জারণ মরিচা পড়া ধীরগতি বিক্রিয়া বছর সম পক্ষান্ত আগুন সেলুলোজের দহনও বিক্রিয়া সেকেন্ডেরও ভগ্নাংশ পরিমাণ সম সংঘটিত বেশিরভাগ বিক্রিয়া ক্ষেত্রে বিক্রিয়া এগিয়ে বিক্রিয়া হার হ্রাস পায় সম ঘনমাত্ পরিবর্তন পরিমাপ বিক্রিয়া হার নির্ণয় রাসায়নিক গতিবিদ্যা ভৌত রসায়ন অংশ রাসায়নিক বিক্রিয়া হার পরিমাপ পূর্বানুমান আলোচনা সম্ভাব্য বিক্রিয়া ক্রিয়াকৌশল ধাপ সম্পর্কে ধারণা পেতে বিক্রিয়া হার উপাত্ত কিভাবে লাগানো সেও আলোচনাও রসায়ন প্রকৌশল উৎসেচক তত্ত্ব পরিবেশ প্রকৌশল শাখাত রাসায়নিক গতিবিদ্যা ধারণা প্রয়োগ
