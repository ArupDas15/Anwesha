জাপান ফুটবল অ্যাসোসিয়েশন জাপানিঃ 日本サッカー協会 প্রতিবর্ণী chon Sakkā kai ইংরেজিঃ apan Football Association এছাড়াও সংক্ষেপ জাপান এফএ নামে পরিচিত জাপান ফুটবল সর্বোচ্চ নিয়ন্ত্রক সংস্থা সংস্থা ১৯২১ সাল প্রতিষ্ঠিত প্রতিষ্ঠা বছরে ফুটবল সর্বোচ্চ সংস্থা ফিফা সদস্যপদ লাভ পাশাপাশি প্রতিষ্ঠা ৩৩ বছর ১৯৫৪ সাল সংস্থা আঞ্চলিক সংস্থা এএফসি সদস্যপদ লাভ সংস্থার সদর দপ্ত জাপান রাজধানী টোকিওতে অবস্থিত সংস্থা জাপান পুরুষ নারী অনূর্ধ্ব ২৩ দল পাশাপাশি ঘরোয়া ফুটবল লীগ ডাব্লিউই লীগ প্রতিযোগিতা কার্যক্রম পরিচালনা বর্তমানে জাপান ফুটবল অ্যাসোসিয়েশন সভাপতি দায়িত্ব পালন কাজো তাশিমা সম্পাদক দায়িত্ব পালন কিয়োটাকা সুহা
