ফরাসি ফুটবল ফেডারেশন ফরাসিঃ Fédération Française de Football ইংরেজিঃ bench Football keerat এছাড়াও সংক্ষেপ এফএফএফ নামে পরিচিত ফ্রান্স ফুটবল সর্বোচ্চ নিয়ন্ত্রক সংস্থা বিদেশী বিভাগ গুয়াদেলুপ ফরাসি গুইয়ানা মার্তিনিকু মায়োত্ত রিইউনিয়ন বিদেশী দল নিউ ক্যালেদোনিয়া ফরাসি পলিনেশিয়া ওয়ালিস ফুটুনা সেন্ট পিয়ে মিকেলন সেন্ট বার্থলেমি সেন্ট মার্টিন মোনাকো সংস্থা ১৯১৯ সাল ৭ই এপ্রিল তারিখে প্রতিষ্ঠিত ১৯০৭ সাল ইউএসএফএস পরিবর্তে ফিফায় যোগ দিয়েছিল প্রতিষ্ঠাতা সদস্য অন্যদিকে প্রতিষ্ঠা ৩৫ বছর ১৯৫৪ সাল সংস্থা আঞ্চলিক সংস্থা উয়েফা সদস্যপদ লাভ সংস্থার সদর দপ্ত ফ্রান্স রাজধানী প্যারিস অবস্থিত সংস্থা ফ্রান্স পুরুষ নারী অনূর্ধ্ব ২৩ দল পাশাপাশি ঘরোয়া ফুটবল লীগ ১ লীগ ২ কুপ দে ফ্রান্স প্রতিযোগিতা কার্যক্রম পরিচালনা বর্তমানে ফরাসি ফুটবল ফেডারেশন সভাপতি দায়িত্ব পালন নোয়েল ল গ্রাত সম্পাদক দায়িত্ব পালন লো গেওর্গেস
