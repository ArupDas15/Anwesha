ব্যাটিং অর্ডার ইংরেজিঃ datti order বিশ্ব অন্যতম জনপ্রি খেলা ক্রিকেট পরিভাষা ব্যাটসম্যান ব্যাট পর্যায়ক্রম ব্যাটিং উদ্বোধন শীর্ষ পর্যা মাঝারি পর্যা স্তর নিম্ন পর্যা ব্যাপ্তি দল ইনিংস ব্যাটসম্যানদ ধারাবাহিকভাবে খেলা অংশগ্রহণ প্রক্রিয়াবিশেষ ইনিংস সর্বদাই দুইজন ব্যাটসম্যান সম অবস্থান দল ইনিংস সম্পন্ন লক্ষ্য এগারো খেলোয়াড় সকলকেই ব্যাট প্রয়োজন পড় ব্যতিক্রম ডিক্লেয়া কারণে ইনিংস সমাপ্তি
