গাণিতিক ভাগ প্রক্রিয়া ভাজক হর শুন্য ব্যবহার শূন্য ভাগ গাণিতিকভাবে ভাগ 0  প্রকাশ ভাজ্য লব যাক ≠ 0 সংখ্যা সম্ভব 0 গুণ অশূন্য সংখ্যা একারণ পাটিগণিতে উক্ত রাশি অর্থহীন সংখ্যা শূন্য বিভাজন অসংজ্ঞায়িত এছাড়াও সংখ্যা শূন্য গুণ গুণফল শূন্য 0 0 রাশিমালাটিও অসংজ্ঞায়িত সীমা আকার প্রকাশ অনির্ধার্য গঠন ঐতিহাসিকভাবে প্রাচীন লেখালেখিসমূহ ১৭৩৪ সাল প্রকাশিত জর্জ বার্কলি দ্য অ্যানালিস্ট গোস্টস অব ডিপার্টেট কোয়ান্টিটিস গ্রন্থ ক্ষুদ্রাতিক্ষুদ্ পরিমাপ নির্ণয় সম্পর্কিত আলোচনা ক্যালকুলাস গাণিতিকভাবে 0 মান নির্ধারণ অসম্ভাবনা উল্লেখ লেখা 0 অসংজ্ঞায়িতরেইম্যানী গোলক প্রক্ষিপ্তভাবে সম্প্রসারিত বাস্তব রেখা ন্যায় গাণিতিক ব্যবস্থাপনা furniture 0 সংজ্ঞায়িত সত্ত্বেও এধরন গাণিতিক পরিকাঠামো পাটিগণিত প্রতিটি সূত্র গাণিতিক ক্ষেত্র স্বীকার্য সন্তোষজনক কম্পিউটিং শূন্য ভাগ চেষ্টা প্রোগ্রাম ভুল ঘটতে প্রোগ্রামিং পরিবেশ শূন্য বিভাজিত সংখ্যা নির্ভর IEEE 754 ফ্লোটিং পয়েন্ট স্টান্ডার্ড আলো ধনাত্মক ঋণাত্মক অসীমতা তৈরি এক্সেপশন এরর মেসেজ তৈরি প্রোগ্রাম ধ্বংস minati প্রকার NaN মান আবির্ভাব ঘটায় অসীম সংখ্যক লুপ ক্রাশ কম্পিউটার হ্যাং ফেলে
