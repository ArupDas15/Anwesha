নৃত্য পরিচালনা ইংরেজিঃ autograph শারীরিক দেহ গতিবিধি প্রতিকৃতি ক্রম নকশা শিল্প অনুশীলন গতি রূপ উভয়ই নির্দিষ্ট কোরিওগ্রাফিকে পরিচালনা নকশা উল্লেখ নৃত্য পরিচালক হলেন ব্যক্তি নৃত্য শিল্প অনুশীলন নৃত্য পরিচালনা নৃত্য পরিচালনা ব্যাল অপেরা গীতি নাট্য চিয়ারলিডিং সিনেমাটোগ্রাফি জিমন্যাস্টিকস পোশাক প্রদর্শনী বরফ স্কেটিং মার্চিং ব্যান্ড শো কোয়া মঞ্চনাট্য সিঙ্ক্রোনাইজড সাঁতার কার্ডিস্ট্রি ভিডিও গেম প্রোডাকশন অ্যানিমেটেড আর্টসহ ব্যবহৃত পরিবেশন শিল্পকলা নৃত্য পরিচালনা মানব আন্দোলন রূপ প্রযোজ্য নৃত্য নৃত্য পরিচালনা নৃত্য রচনা হিসেবেও পরিচিত
