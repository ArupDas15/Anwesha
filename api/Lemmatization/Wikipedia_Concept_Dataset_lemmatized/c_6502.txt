আইফোন এক্স আইফোন এক্স অ্যাপল ইনকর্পোরেটেড নকশা তৈরি বাজারজাত স্মার্টফোন ২০১৭ সাল ১২ সেপ্টেম্বর যুক্তরাষ্ট্ ক্যালিফোর্নিয়া অবস্থিত অ্যাপল পার্ক ক্যাম্পাস স্টিভ জবস মিলনায়তন আইফোন ৮ আইফোন ৮ প্লাস স্মার্টফোন আইফোন এক্স উন্মোচন অ্যাপল প্রধান নির্বাহী কর্মকর্তা টিম কুক ২০১৭ সাল ৩ নভেম্বর আনুষ্ঠানিক স্মার্টফোন বাজার অ্যাপল ধারাবাহিক মডেল এক্স মডেল স্মার্টফোন বাজার সর্বশেষ প্রযুক্তি সুবিধা যুক্ত আইওএস ১১ অপারেটিং সিস্টেম চালিত স্মার্টফোন থাকছে বাহ্যিক বোতাম ওএলইডি প্রযুক্তি পর্দা ডুয়েল ক্যামেরা যুক্ত ফেস রিকগনিশন সুবিধা ফেস আইডি নামের আনলক পদ্ধতি
