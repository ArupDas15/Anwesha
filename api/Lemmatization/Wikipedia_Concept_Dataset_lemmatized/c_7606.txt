কোকেইন কোকা গাছ নির্যাস উপস্থিত উপক্ষা উত্তেজক মাদক দ্রব্য কোকা গাছ বীজ পাতা প্রস্তুত কোকেইন প্রস্তুতিত যায়ঃ বাটা পেস্ট ৪০ ৮০ কোকেইন সালফেট অন্যান্য অশুদ্ধি কোকেইন হাইড্রোক্লোরাইড লবণ জল দ্রাব্য শুদ্ধ সর্বাধিক ব্যবহৃতমুক্ত ক্ষার ফ্রি বেস কোকেইন হাইড্রোক্লোরাইড নিষ্কাষিত জল অদ্রাব্য খাওয়া ইঞ্জেক্সন শক্ত ধূমপান সহজ ক্র্যাক অশুদ্ধ মুক্ত ক্ষার উদ্বাইয়ী কেলাস জল ডেলা অবস্থা গরম জল ফোটবা শব্দ নামে ক্র্যাক
