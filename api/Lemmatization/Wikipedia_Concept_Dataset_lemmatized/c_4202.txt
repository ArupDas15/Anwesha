ব্যাপক প্রতিস্থাপন ইংরেজিঃ The great cement ফরাসিঃ le granh cement দক্ষিণপন্থী ষড়যন্ত্র তত্ত্ব অনুসারে শ্বেতাঙ্গ ক্যাথোলিক ফরাসী জনগণ শ্বেতাঙ্গ খ্রিস্টান ইউরোপী জনগণ প্রচুর পরিমাণ পদ্ধতিগতভাবে অ ইউরোপী জনগণের প্রতিস্থাপিত অ ইউরোপী জনগণের বিশেষভাবে আরব বার্বা মধ্যপ্রাচ্য আফ্রিকীয় সাহা দক্ষিণ আফ্রিকীয় কৃষ্ণাঙ্গ জনগণ প্রতিস্থাপন ঘটছে গণ অভিবাসন উচ্চ জন্মহার জনসংখ্যা বৃদ্ধি ফ্রান্স মুসলিমদ উপস্থিতি সম্পর্কিত সম্পর্কিত ফরাসী সংস্কৃতি সভ্যতা সম্ভাব্য বিপদ ধ্বংস ষড়যন্ত্র তত্ত্ব সাধারণভাবে বৈশ্বিক উদারপন্থী সম্ভ্রান্তদ দায়ী ব্রাসেলস ইউরোপী ইউনিয়ন অভিযোগ ইচ্ছাকৃতভাবে ইউরোপ ইউরোপীয়দ প্রতিস্থাপিত ষড়যন্ত্র তত্ত্ব জনপ্রি রেনো ক্যামু tena amul প্রতিস্থাপন শ্বেতাঙ্গ গণহত্যা ধারণা পাশ্চাত্য অতি দক্ষিণপন্থি আন্দোলনেই অলঙ্করণ ব্যবহৃত ধারণাটি প্রধান প্রচারক কেবল দক্ষিণপন্থী নন প্রচার জড়িত বিস্তৃত পরিসর প্রতিবাদী আন্দোলন যেমনঃ পেগিডা মতাদর্শগত দলী আন্দোলন যেমনঃ ল ইডেন্টিটেয়া Les Identitaires ব্লগা ফিয়র্দম্যান jorda লরেন সাউদার্ন Lauren muthe বুদ্ধিজীবীরাও যেমনঃ এরিক জেমোউ উল্লেখযোগ্য দক্ষিণপন্থী ওয়েবসাইট গেটস অফ ভিয়েনা gate ghenna পলিটিকালি ইনকারেক্ট Politically Incorrect Fdesouche ব্লগারদ ষড়যন্ত্র তত্ত্বটিকে ছড়ি জনপ্রি
