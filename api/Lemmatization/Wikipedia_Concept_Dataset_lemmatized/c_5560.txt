ঐশ্বরিয়া মজমুদা গুজরাটিঃઐશ્વર્યા મજમુદાર জন্মঃ ৫ অক্টোবর ১৯৯৩ ভারত গুজরাট রাজ্যের আহমেদাবাদ শহর কণ্ঠশিল্পী ২০০৭ ২০০৮ বাদ্যযন্ত্র সত্যিকা প্রদর্শনী স্টার ভয়েস অফ ইন্ডিয়া ছোট উস্তাদ ১৫ বছর বয়সে বিজয়ী জনপ্রিয়তা অর্জন পারফরমেন্স সুখ্যাতি বিচারকদ প্রতিযোগিতা আনোশা দত্ত গুপ্ত প্রতিযোগিতা প্রতিযোগিতা জিতেছিলেন হিমেশ রেশমিয়া হিম্স ওয়ারিয়র্স দল মিউজিক কা মহা মোকাবেলাত অংশ নেন গুজরাটি হিন্দি চলচ্চিত্র গান গেয়েছেন অন্তাকশারী গ্রেট চ্যালেঞ্জ অংশ নিয়েছিলেন
