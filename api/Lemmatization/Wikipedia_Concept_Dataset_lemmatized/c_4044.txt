বনভূমি ইংরেজিঃ biddlan কম ঘনত্ব প্রচুর সূর্যালোক সীমিত ছায়া উন্মুক্ত আবাস গঠন কাষ্ঠভূমি ঘাসসহ গুল্ম বিরুৎজাতীয় উদ্ভিদ বনতল আন্ডারস্টোরি আশ্রয় সক্ষম শুষ্ক অবস্থা প্রাইমারি সেকেন্ডারি ক্রমাগমণ পর্যা কাষ্ঠভূমি গুল্মভূমি ক্রান্তিকাল অতিক্রম বন কাষ্ঠভূমি পার্থক্য বন বৃক্ষ ঘনত্ব আয়তন ঠাস বুনট চাঁদোয়া বিস্তৃত অবিচ্ছিন্ন ছায়া প্রদান বন কাষ্ঠভূমি উচ্চতা কম কাষ্ঠভূমি বনতল মাটি বৈশিষ্ট্য পরিবর্তীত কোনোটি গুল্মপূর্ণ কোনোটি উন্মুক্ত ঘাসপ্রাচুর্য বনতল সংরক্ষণবিদ কাষ্ঠভূমি সংরক্ষণ কঠোর পরিশ্রম উদাহরণস্বরূপ উত্তরপশ্চিম ইন্ডিয়ানা কাষ্ঠভূমি ইন্ডিয়ানা বালিয়াড়ি অংশ অক্ষুণ্ণ ব্রিটিশ কাষ্ঠভূমি ব্যবস্থাপনা উডল্যান্ড বৃক্ষ আবৃত ঘন যেকোনো ক্ষুদ্র অঞ্চল বুঝানো ব্রিটিশ দ্বীপপুঞ্জ শুধুমাত্র ব্যাপক জায়গাজুড়ে বিস্তৃত অঞ্চল ঘনত্বনির্বিশেষে রাজকী বন এগুলো বৃক্ষপূর্ণ বুঝায় প্রাচীণ উডল্যান্ড শব্দ ব্রিটিশ প্রকৃতি সংরক্ষণ যেকোনো বৃক্ষপূর্ণ ভূমি বুঝায় দীর্ঘ সম বিদ্যমান আমেরিকান শব্দগুচ্ছ ওল্ড গ্রোথ সমার্থক উডলট সম্বন্ধযুক্ত আমেরিকান শব্দ জ্বালানি ব্যবহৃত বৃক্ষ স্ট্যান্ড বুঝায় প্রায়োগিকভাবে উডলট ঠাস বুনট চাঁদোয়া এতোই ছোটো আকার কিনার অনুপ্রবেশকৃত আলো পরিবেশগতভাবে বন কাষ্ঠভূমি সম্পর্কিত
