কলকাতা শেয়া বাজার দি ক্যালকা স্টক এক্সচেঞ্জ পোষাকি নামঃ দি ক্যালকা স্টক এক্সচেঞ্জ অ্যাসোসিয়েশন লিমিটেড কলকাতা শহর স্টক এক্সচেঞ্জ লায়ন্স রেঞ্জ অবস্থিত ১৯০৮ সাল স্টক এক্সচেঞ্জটি ভারত দ্বিতীয় বৃহত্তম শেয়া বাজার ১৮৩০ সাল কলকাতা নিম গাছ তলায় শেয়া বাজার কাজকর্ম চলত ভারত সবচেয়ে পুরনো সিকিউরি ডিলিং ব্রিটিশ ইস্ট ইন্ডিয়া কোম্পানি লোন সিকিউরি ১৯০৮ সাল ১৫০ সদস্য শেয়া বাজার লাইসনস রেঞ্জ বর্তমান শেয়া বাজার ভবন নির্মিত ১৯২৮ সাল কলকাতা শেয়া বাজার ১৯৫৬ সাল সিকিউরি কনট্র্যাকটস রেগুলেশন অ্যাক্ট আওতাভুক্ত বিষ ভারত সরকার অনুমোদন পায় ১৯৮০ সাল ১৪ এপ্রিল ১৯৯৭ সাল শেয়া কেনাবেচার নিয়মেই কলকাতা শেয়া বাজার চলত ১৯৯৭ সাল স্টার সিএসই স্ক্রিন বেসড ট্রেডিং অ্যান্ড রিপোর্টিং নামে বাণিজ্য ব্যবস্থা সূত্রপাত ঘটে বম্বে স্টক এক্সচেঞ্জ বিএস ৫ শতাংশ শেয়া বিনিম কলকাতা শেয়া বাজার কৌশলগত বিনিয়োগ
