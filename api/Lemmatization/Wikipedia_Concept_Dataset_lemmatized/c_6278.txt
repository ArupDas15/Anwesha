তন্দ্রাবিলাস হুমায়ূন আহমেদ মিসি আলি চরিত্রভিত্তিক উপন্যাস ১১ নম্বর উপন্যাস প্রকাশিত ১৩ জানুয়ারি ১৯৯৭ সাল স্বত্বাধিকারী লেখক স্ত্রী গুলতেকিন আহমেদ প্রকাশিত দিব্য প্রকাশনী মুদ্রিত মূল্য ৫৫ টাকা প্রচ্ছদ অঙ্কন ধ্রুব এষ তন্দ্রাবিলাস উপন্যাস মানুষ অবচেতন মন পরাবাস্তব জগৎ অদ্ভুত মেলবন্ধন পুরো উপন্যাসটিতেই হুমায়ূন আহমেদ মিসি আলি চরিত্র মানুষ অদ্ভুত আচরণ অস্বাভাবিক ক্ষমতা যৌক্তিক উপস্থাপন
