ইরাক ফুটবল অ্যাসোসিয়েশন আরবিঃ الاتحاد العراقي لكرة القدم‎‎ ইংরেজিঃ Iraq Football Association এছাড়াও সংক্ষেপ আইএফএ নামে পরিচিত ইরাক ফুটবল সর্বোচ্চ নিয়ন্ত্রক সংস্থা সংস্থা ১৯৪৮ সাল প্রতিষ্ঠিত প্রতিষ্ঠা ২ বছর ফুটবল সর্বোচ্চ সংস্থা ফিফা সদস্যপদ লাভ পাশাপাশি প্রতিষ্ঠা ২২ বছর ১৯৭০ সাল সংস্থা আঞ্চলিক সংস্থা এএফসি সদস্যপদ লাভ সংস্থার সদর দপ্ত ইরাক রাজধানী বাগদাদ অবস্থিত সংস্থা ইরাক পুরুষ নারী অনূর্ধ্ব ২৩ দল পাশাপাশি ঘরোয়া ফুটবল ইরাকী প্রিমিয়া লীগ ইরাকী নারী প্রিমিয়া লীগ ইরাক এফএ কাপ প্রতিযোগিতা কার্যক্রম পরিচালনা বর্তমানে ইরাক ফুটবল অ্যাসোসিয়েশন সভাপতি দায়িত্ব পালন ইয়াদ আল নাদাউই সম্পাদক দায়িত্ব পালন মুহাম্মদ ফারহান ওবাইদ
