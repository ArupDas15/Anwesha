সার্বজনিন ডকুমেন্টেশন পদ্ধতি ইংরেজিঃ The système universitaire de documentation SUDOC ফ্রান্স বিশ্ববিদ্যাল উচ্চশিক্ষা প্রতিষ্ঠান  আওতাধীন দস্তাবেজ সনাক্তকরণ অনুসরণ পরিচালনা পদ্ধতি পদ্ধতিটিত ১৩ মিলিয়ন রেফারেন্স শিক্ষার্থী গবেষকদ ৩৪০০ এরও ডকুমেন্টেশন সেন্টা গ্রন্থাগার অবস্থান সম্পর্কিত তথ্য অনুসন্ধান অনুমতি উচ্চশিক্ষা বাইবেলোগ্রাফিক সংস্থা এবিইএস রক্ষণাবেক্ষণ
