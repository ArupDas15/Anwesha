উইন্ডোজ এক্সপি hindola XP মাইক্রোসফট কর্পোরেশন তৈরি অপারেটিং সিস্টেম জনপ্রি অপারেটিং সিস্টেম মাইক্রোসফট উইন্ডোজ এক্সপি সবচেয়ে এগিয়ে লিনাক্স উইন্ডোজ উম্মুক্ত সহজ জনপ্রি অপারেটিং সিস্টেম জনপ্রি অপারেটিং সিস্টেম মাইক্রোসফট উইন্ডোজ সাধারণত সিডি ডিভিডি ইনস্টল সিডি ডিভিডি রম থাকলে ইনস্টল ইউএসবি ডিভাইস ফ্লাশ ডিক্স মেমোরী কার্ড বাংলাদেশ এখনো ৯০ কম্পিউটার ব্যবহারকারী অপারেটিং সিষ্টেম উইন্ডোজ এক্সপি পছন্দ বাজার ফিচা সম্পন্ন অপারেটিং সিষ্টেম ভিস্তা সেভেন উবুন্টু ডেবিয়ান ম্যানড্রেক ম্যাক মূল এক্সপি বান্ধব উইন্ডোজ এক্সপি উইন্ডোজ ভিস্তা উইন্ডোজ ৭ বাজার এসেছে উইন্ডোজ এক্সপি ব্যবহারকারী কমে যায়নি উইন্ডোজ পরবর্তী সংস্করণ গুলোও জনপ্রিয়তা পায়নি উইন্ডোজ ৭ ব্যতীত
