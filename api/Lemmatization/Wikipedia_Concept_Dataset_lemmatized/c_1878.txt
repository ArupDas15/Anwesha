যৌগ অষ্টক পূর্ণতা কম সংখ্যক ইলেকট্রন যোজ্যতা স্তর ধারণ ইলেকট্রন ঘাটতি বন্ধন প্রায়শই ৩ কেন্দ্রীক ২ ইলেকট্রন বন্ধন বোরন যৌগে ঘটনা পরিলক্ষিত ইলেকট্রন ঘাটতি শব্দ প্রায়শ জৈব রসায়ন পাই বন্ধন বোঝাতে ব্যবহৃত অ্যালকেন অ্যারিন প্রভৃতিত পোলারায়ন ঘটে এছাড়া নাইট্রোবেনজিন অ্যাক্রোলোনাইট্রাইল যৌগেও কারণে ইলেকট্রন সাধারণত কার্বন কার্বন দ্বিবন্ধন C C নিউক্লিওফিলিক চরিত্র গেলেও এক্ষেত্রে ইলেকট্রন ঘাটতি কারণে ইলেকট্রোফিলিক চরিত্র অধিকারী নিউক্লিওফাইল আক্রমণ শিকার যেমনটা মাইকেল সংযোজন নিউক্লিওফিলিক অ্যারোমেটিক প্রতিস্থাপন বিক্রিয়া উচ্চ পোলারায়িত যৌগ বোরোন ট্রাইফ্লোরাইড সালফা টেট্রাফ্লোরাইড যৌগে ইলেকট্রন ঘাটতি থাকা কারণে লুইস এসিড প্রবণতা দেখায়
