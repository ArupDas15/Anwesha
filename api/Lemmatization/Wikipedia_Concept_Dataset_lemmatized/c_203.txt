সিপাহি মঙ্গল পাণ্ডে হিন্দিঃ मंगल पांडे শুনুন ১৯ জুলাই ১৮২৭ ৮ এপ্রিল ১৮৫৭ ভারতী সৈনিক ১৮৫৭ সাল ভারতী সিপাহী বিদ্রোহ সূচনা মূল ভূমিকা পালনকারী ব্রিটিশ ইস্ট ইন্ডিয়া কোম্পানি ৩৪তম বেঙ্গল নেটিভ ইনফ্যান্ট্রি বিএনআ সৈন্য রেজিমেন্ট সিপাহী সমকালীন ব্রিটিশ মতামত বিশ্বাসঘাতক বিদ্রোহী নিন্দা করলেও মঙ্গল পাণ্ডে আধুনিক ভারত নায়ক ১৯৮৪ সাল ভারত সরকার স্মরণ ডাকটিকিট জারি করেছিল একাধিক চলচ্চিত্র জীবন ক্রিয়াকলাপ চিত্রিত
