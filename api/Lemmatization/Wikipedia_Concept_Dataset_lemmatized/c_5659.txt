ন্যায্য মার্কিন যুক্তরাষ্ট্ আইন মতবাদ প্রথমে মেধাস্বত্বধারী অনুমতি মেধাস্বত্বযুক্ত উপাদান সীমিত ব্যবহার অনুমতি মেধাস্বত্ব সীমিত ব্যবহার লক্ষ্য প্রণীত পদ্ধতি বিস্তৃত বিতরণ জনস্বার্থ মেধাস্বত্বধারীদ অধিকার সৃজনশীল কাজের ব্যবহার ভারসাম্য রাখে মেধাস্বত্ব লঙ্ঘন প্রতিরক্ষা নির্দিষ্ট সীমিত ব্যবহার অনুমতি অন্যথায় মেধাস্বত্ব লঙ্ঘন বিবেচিত
