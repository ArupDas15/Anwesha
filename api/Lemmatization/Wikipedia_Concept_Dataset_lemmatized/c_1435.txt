ইন্টারনেট সংরক্ষণাগার বৈদ্যুতিক তথ্য উপাত্ত দূরবর্তী স্থান অবস্থিত তৃতীয় পক্ষ গচ্ছিত রাখা সেবাকে বোঝা সেবা গ্রহণ ইন্টারনেট সংযুক্ত চিরায়ত স্থানী সংরক্ষণাগার ব্যবস্থা ডিস্ক বহনযোগ্য সংরক্ষণাগার ব্যবস্থা অপ্টিকাল মাধ্যম ফ্ল্যাশ ড্রাইভ ইত্যাদি বিকল্প ইন্টারনেট সংরক্ষণাগার ইংরেজিত প্রায়শ ক্লাউড স্টোরেজ ক্লাউড সংরক্ষণাগারও উপাত্ত সংরক্ষণ মডেলটিত ডিজিটাল উপাত্তসমূহ লজিক্যাল পুলস একাধিক সার্ভা ফিজিক্যাল স্টোরেজ সংরক্ষিত ফিজিক্যাল পরিবেশ সাধারণত হোস্টিং কোম্পানি মালিকানা পরিচালনা এসব ক্লাউড স্টোরেজ প্রদানকারী উপাত্ত সহজলভ্য প্রবেশ নিশ্চিত ফিজিক্যাল পরিবেশ সুরক্ষা চালিত ব্যবহারকারী সংস্থা ব্যবহারকারী সংস্থা অ্যাপ্লিকেশন উপাত্ত সংরক্ষণ স্টোরেজ ক্যাপাসি তথ্য ধারণ ক্ষমতা ক্রয় ইজারা নেয় ক্লাউড স্টোরেজ সেবা অবস্থিত ক্লাউড কম্পিউটার সেবা ওয়েব সার্ভিস অ্যাপ্লিকেশন প্রোগ্রামিং ইন্টারফেস এপিআই এপিআই অ্যাপ্লিকেশনস সমূহ ক্লাউড ডেক্সটপ স্টোরেজ ক্লাউড স্টোরেজ গেটওয়ে ওয়েব বেসড কনটেন্ট ম্যানেজম্যান্ট সিস্টেম অ্যাক্সেস
