শীর্ষ সাংবিধানিক আদালত আরবিঃ المحكمة الدستورية العليا‎‎ উচ্চারণঃ আল মাহকামাহ আল দুষ্টরিয়া আল উলিয়া সিরিয়া আরব প্রজাতন্ত্র সর্বোচ্চ বিচার বিভাগীয় কর্তৃপক্ষ সুপ্রিম কোর্ট নির্বাচনী বিরোধ নিষ্পত্তি প্রধানমন্ত্রী গণ কাউন্সিল কর্তৃক আপত্তিকৃত আইন সাংবিধানিকতা বিষ রা বিল ডিক্রি বিধিবিধান সংবিধানিকতা সম্পর্কে মতামত ১৯৭৩ সাল সংবিধান অধীন আদালত প্রতিষ্ঠিত প্রজাতন্ত্র রাষ্ট্রপতি প্রজাতন্ত্র জনপ্রি গণভোট প্রণীত সংশোধিত আইন বৈধতা প্রশ্ন তোলা উচ্চ সাংবিধানিক আদালতের নিষিদ্ধ আদালত রাষ্ট্রপতি বিচারক গঠিত বছরের পুনর্নবায়নযোগ্য মেয়াদ নিযুক্ত
