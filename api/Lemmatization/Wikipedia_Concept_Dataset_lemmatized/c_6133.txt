লেখা সম্পাদক টেক্সট এডিটর text sitor শ্রেণীর কম্পিউটার প্রোগ্রাম টেক্সট ফাইল সম্পাদনা মাইক্রোসফ্ট নোটপ্যাড নামকরণ অনুসরণ জাতীয় প্রোগ্রাম সম নোটপ্যাড সফ্টওয়্যা নামে পরিচিত অপারেটিং সিস্টেম সফ্টওয়্যা ডেভলপমেন্ট প্যাকেজ লেখা সম্পাদক অন্তর্ভুক্ত কনফিগারেশন ফাইল সাহায্যমূলক ফাইল প্রোগ্রামিং ভাষার সোর্স কোড ফাইল পাঠ সম্পাদনা ব্যবহৃত
