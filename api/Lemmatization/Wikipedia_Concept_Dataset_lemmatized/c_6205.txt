আন্তর্জাতিক মালিকানাবিহীন নাম International Nonproprietary Name সংক্ষেপ এন এন INN আরো নাম রিকমেন্ডেড ইন্টারন্যাশ্ন্যাল ননপ্রোপ্রাইটরি ন্যাম recommended International Nonproprietary Name সুপারিশকৃত আন্তর্জাতিক মালিকানাবিহীন নাম সংক্ষেপ এন এন rINN প্রৌপোউড্স ইন্টারন্যাশ্ন্যাল ননপ্রোপ্রাইটরি ন্যাম proposed International Nonproprietary Name প্রস্তাবকৃত আন্তর্জাতিক মালিকানাবিহীন নাম সংক্ষেপ এন এন pINN হিসেবেও বিশ্ব স্বাস্থ্য সংস্থা ফার্মাসি্উটিক্যাল উপাদান সমূহ শনাক্তকরণ নাম প্রদান মালিকানাধীনদ প্রস্তুতকৃত দ্রব্য প্রদিত উপাদান নামকরণ কার্যকরী উপাদান শনাক্তকরণ বিশৃঙ্খলা এন এন নামসমূহ পৃথক স্বতন্ত্রভাবে প্রতিটি উপাদান যোগাযোগ রক্ষা বিশৃঙ্খলাবিহীনভাবে ব্যবস্থাপত্ সহজে উপস্থাপন সুবিধা প্রদান নাম জেনেরিক নামেও আখ্যায়িত যেমনটি রসায়ন আইইউপিএসি নামকরণ বিশ্ব স্বাস্থ্য সংস্থা এন এন নামসমূহ ইংরেজি ল্যাটিন ফ্রেঞ্চ রাশিয়ান স্প্যানিশ ভাষা আরবি চাইনিজ সংস্করণে প্রকৃত নকশায় অন্তর্ভুক্ত সংকলিত
