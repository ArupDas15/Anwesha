পদার্থবিজ্ঞান অদিক রাশি ইংরেজিঃ kalar jhanti রাশিসমূহ যেগুলোকে প্রকাশ সদিক রাশি নির্দিষ্ট দিক উল্লেখ প্রয়োজন কেবল মান দ্বারাই যেমনঃ শুধুমাত্র সংখ্যা এদেরকে প্রকাশ অদিক রাশি ধারণা গণিতেও অদিক রাশি অপর নাম স্কেলা রাশি অদিক রাশি উদাহরণ তাপমাত্রা প্রকাশ মান ১০ কেলভিন ১০ ডিগ্রী সেলসিয়াস কেবল উল্লেখ দিক অপরদিকে সদিক রাশি বেগ প্রকাশ মান দ্বিমাত্রিক ত্রিমাত্রিক পোলা স্থানাঙ্ক উল্লেখ্য প্রয়োজন পদার্থবিজ্ঞান অদিক রাশি দুরত্ব দ্রুতি ভর সম তাপমাত্রা শক্তি
