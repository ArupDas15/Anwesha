ওমেন্স টেনিস অ্যাসোসিয়েশন ইংরেজিতেঃ rome tennis Association সংক্ষেপ ডব্লিউটিএ WTA ১৯৭৩ সাল বিলি জিন কিং কর্তৃক প্রতিষ্ঠিত পেশাদা মহিলা টেনিস খেলোয়ারদ সংস্থা ডব্লিউটিএ ট্যু আয়োজন ডব্লিউটিএ ট্যু বিশ্বব্যাপী অনুষ্ঠিত মহিলা টেনিস খেলোয়াড়দ টেনিস চ্যাম্পিয়নশিপ সংস্থা আয়োজন ১৯৭৩ সাল জুন ডব্লিউটিএ প্রতিষ্ঠা ডব্লিউটিএ ইতিহাস সূচনা টেক্সাস হিউস্টন টেনিস তারকা গ্ল্যাডিস হেল্ডম্যান ১৯৭০ সাল সেপ্টেম্বরে ভার্জিনিয়া স্লিমস টুর্নামেন্ট আয়োজন ডব্লিউটিএ প্রধান দপ্ত ফ্লোরিডা সেন্ট পিটার্সবার্গ অবস্থিত ইউরোপিয়ান দপ্ত লন্ডন এশিয়া প্রশান্ত মহাসাগরী দপ্ত বেইজিং অবস্থিত
