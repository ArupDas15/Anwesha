প্রোগ্রামিং ভাষা ইংরেজি ভাষায়ঃ program langhan কৃত্রিম ভাষা যন্ত্র প্রধানত কম্পিউটারের আচরণ নিয়ন্ত্রণ মানুষ মুখের ভাষা প্রোগ্রামিং ভাষাগুলিরও নিজস্ব ব্যাকরণ তথ্য সুবিন্যস্তকরণ প্রক্রিয়াকরণ অ্যালগোরিদমসমূহ নির্ভুলভাবে প্রকাশ প্রোগ্রামিং ভাষা লেখক প্রোগ্রামিং ভাষা কেবল ভাষা বোঝান যেগুলো সম্ভাব্য অ্যালগোরিদম প্রকাশ সক্ষম কখনো কখনো সরল কৃত্রিম ভাষা প্রোগ্রামিং ভাষা কম্পিউটার ভাষা compit langhan প্রোগ্রামিং ভাষা সৃষ্টি সর্বদাই নিত্য প্রোগ্রামিং ভাষা উদ্ভাবিত চলেছে
