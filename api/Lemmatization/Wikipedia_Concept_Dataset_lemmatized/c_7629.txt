নিদানবিজ্ঞান রোগ নির্ণয় বিজ্ঞান ইংরেজিঃ Etiology Aetiology চিকিৎসাবিজ্ঞান শাখা বোঝা রোগ রোগ শ্রেণীর কারণসমূহ গবেষণা বাংলা ভাষা রোগ মূল কারণ নিদান ইংরেজিত কারণকেও এটিওলজি Etiology রোগ নির্ণয় সম্ভব সেটিকে স্বয়ম্ভূত রোগ dopati sease রোগবিস্তা বিজ্ঞান আলোচনা ৬ নির্দেশ এগুলি হলঃউৎপাদক Production Cause যেসব শর্ত রোগ প্রাদুর্ভাব ঘটাতে মূল ভূমিকা পালন আবশ্যক Necessary cause যেসব শর্ত অনুপস্থিতিত রোগ ঘটা সম্ভব এইচআইভি ভাইরাস সংক্রমণ এইডস আবশ্যক Sufficient cause শর্ত পূরণ নিশ্চিতভাবে রোগ সৃষ্টি উপাংশবিশিষ্ট Sufficient companvit cause উপাংশবিশিষ্ট একাধিক উপাংশ গঠিত কোনওটিই এককভাবে রোগ সৃষ্টি একত্রিত রোগ সৃষ্টি অবশ্যম্ভাবী সম্ভাবনাভিত্তিক Probabilistic cause সম্ভাবনাভিত্তিক রোগ সৃষ্টি হবা সম্ভাবনা বৃদ্ধি কারণ আবশ্যক প্রয়োজন আবশ্যক কারণ পরিণামবাদী প্রকৃতি রোগ সৃষ্টি সম্ভাবনা ১ উন্নীত অন্যদিকে আবশ্যক রোগ সৃষ্টি সম্ভাবনা ০ পারবে বিকল্প বাস্তব Counterfactual cause বিকল্প বাস্তব কাল্পনিক যেটি উপস্থিত অনুপস্থিত থাকলে রোগ হবা সম্ভাবনা বাড়তে কমতে রোগ কারণ স্তর থাকতে আণবিক জৈবিক ব্যক্তিগত সামাজিক স্তরের কারণসমূহ রোগ সৃষ্টি রোগ সম্ভাবনা বৃদ্ধি এগুলির কোন্টিকে অগ্রাধিকা ব্যক্তিভেদ ভিন্ন ভিন্ন রোগ কারণ সার্বজনীন কার্যকরী সংজ্ঞা রোগী রোগ কারণ প্রতিমান মডেল নির্মাণ পার্থক্য
