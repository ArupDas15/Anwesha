ডেভিড লুইস নিউম্যান জিন্ম ১১ মার্চ ১৯৫৪ মার্কিন সুরকা সঙ্গীত নির্দেশক ত্রিশ বছরের অধিক সম ১০০ চলচ্চিত্র সুরারোপ আনাস্তাসিয়া ১৯৯৭ চলচ্চিত্র শ্রেষ্ঠ মৌলিক সুর বিভাগ একাডেমি পুরস্কার মনোনয়ন লাভ সর্বাধিক একাডেমি পুরস্কার মনোনীত নিউম্যান পরিবার সঙ্গীত বিভাগ ৯২টি মনোনয়ন লাভ পিতা অ্যালফ্রেড নিউম্যান চাচা এমিল নিউম্যান লাইওনেল নিউম্যান ছোট ভাই টমাস নিউম্যান বোন মারিয়া নিউম্যান চাচাতো ভাই র্যান্ডি নিউম্যান
