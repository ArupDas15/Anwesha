কার্নেল ইংরেজিঃ Kernel কম্পিউটার বিজ্ঞান অপারেটিং সিস্টেম প্রধান উপাদান কম্পিউটার হার্ডওয়া সংগঠিত অ্যাপ্লিকেশন প্রকৃত ডা প্রসেসিং সেতুবন্ধন সিস্টেম সম্পদ সুবিধাসমূহ ব্যাবস্থাপনা কার্নেল দায়িত্ব হার্ডওয়া সফ্টওয়ার উপাদান যোগাযোগ স্বাভাবিকভাবে অপারেটিং সিস্টেম মূল উপাদান কার্নেল সম্পদ সর্বনিম্ন অ্যাবসট্রাকশন লেয়া নিশ্চিত অ্যাপ্লিকেশন সফটওয়া অবশ্যই নিয়ন্ত্রণ কার্নেল ডিজাইন প্রয়োগ ভিত্তিত অপারেটিং সিস্টেম কাজ বিভিন্নভাবে সংগঠিত মনোলিথিক কার্নেল অপারেটিং সিস্টেম কর্মক্ষমতা বৃদ্ধি অপারেটিং সিস্টেম কোড এড্রেস স্পেস সম্পাদনা মাইক্রোকার্নেল বেশিরভাগ অপারেটিং সিস্টেম সার্ভা ইউজা স্পেস কোডগুলো সম্পাদনা অপারেটিং সিস্টেম নিয়ন্ত্রণ ক্ষমতা ভিন্নতা বাড়ানো
