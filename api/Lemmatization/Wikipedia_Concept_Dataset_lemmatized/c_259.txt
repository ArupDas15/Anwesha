ফিলিস্তিন ফুটবল অ্যাসোসিয়েশন আরবিঃ الاتحاد الفلسطيني لكرة القدم‎‎ ইংরেজিঃ paninian Football Association এছাড়াও সংক্ষেপ পিএফএ নামে পরিচিত ফিলিস্তিন ফুটবল সর্বোচ্চ নিয়ন্ত্রক সংস্থা সংস্থা ১৯৬২ সাল প্রতিষ্ঠিত প্রতিষ্ঠা ৩৬ বছর ফুটবল সর্বোচ্চ সংস্থা ফিফা সদস্যপদ লাভ পাশাপাশি বছর সংস্থা আঞ্চলিক সংস্থা এএফসি সদস্যপদ লাভ সংস্থার সদর দপ্ত ফিলিস্তিন জেরুসালেমে অবস্থিত সংস্থা ফিলিস্তিন পুরুষ নারী অনূর্ধ্ব ২৩ দল পাশাপাশি ঘরোয়া ফুটবল গাজা স্ট্রিপ প্রিমিয়া লীগ গাজা স্ট্রিপ লীগ ফিলিস্তিন কাপ প্রতিযোগিতা কার্যক্রম পরিচালনা বর্তমানে ফিলিস্তিন ফুটবল অ্যাসোসিয়েশন সভাপতি দায়িত্ব পালন জিব্রিল আল রাজুব সম্পাদক দায়িত্ব পালন ফিরাস আবু হিলাল
