সিলেট সিটি কর্পোরেশন ২০০১ খ্রিষ্টাব্দ প্রতিষ্ঠিত সিটি কর্পোরেশন সিলেট সিটি কর্পোরেশন আইন ২০০১ প্রতিষ্ঠিত সিটি কর্পোরেশন স্থাপনকল্প উক্ত আইন প্রণীত সিটি কর্পোরেশন এলাকা গণপ্রজাতন্ত্রী বাংলাদেশ সংবিধান ৫৯ অনুচ্ছেদ উদ্দেশ্য পূরণার্থ স্থানী প্রশাসনিক ইউনিট প্রতিষ্ঠিত পরবর্তিকাল আইন ২০০২ সংশোধিত সিলেট সিটি কর্পোরেশন সংশোধন আইন ২০০২
